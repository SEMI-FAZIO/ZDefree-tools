# Test fixtures

Files in this directory are copies from the [ZDefree](https://github.com/SEMI-FAZIO/ZDefree)
repository, used to test the loader and compiler end-to-end.

When ZDefree's `strategies/common/*.json` are updated, mirror them here.
Until CI does this automatically, drift is possible — that's by design for
fast iteration in the early phase.
